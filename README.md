# Pizzabot - Your Personal Pizza Recipe & Chat Assistant

Pizzabot is a Python-powered Discord bot that delivers a seamless,
interactive pizza-making experience.\
From over 10+ curated recipes to an AI-powered pizza-themed chatbot,
users can explore, learn, and cook --- all inside Discord.

This project serves as the **first major portfolio project**, showcasing
backend engineering, API integration, collaboration, and deployment
capability.
